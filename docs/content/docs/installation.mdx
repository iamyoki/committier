## npx

You can directly execute via

```sh tab="npx"
npx committier <command>
```

```sh tab="yarn"
yarn dlx committier <command>
```

```sh tab="pnpm"
pnpm dlx committier <command>
```

```sh tab="bun"
bunx committier <command>
```

## Install Locally (recommended)

```sh tab="npm"
npm install committier -D
```

```sh tab="yarn"
yarn add committier -D
```

```sh tab="pnpm"
pnpm add committier -D
```

```sh tab="bun"
bun add committier -d
```

<Callout>AI is now in Beta, install via `pnpm add committier@beta -D`</Callout>

Optionally add a script in `package.json`, to map a shorter command for commit CLI

```json
{
  "scripts": {
    "commit": "committier commit",
    "commit:ai": "committier ai"
  }
}
```

Then

```sh
pnpm commit
```
